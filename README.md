A full-cycle web application for ordering customized wraps, featuring an interactive front-end ingredient selector and a comprehensive back-end stock management system.

## ✨ Key Features

**Customer Facing (Front-End)**
* **Interactive Wrap Builder:** Users can select their preferred ingredients with a real-time visual preview of their customized wrap.
* **Checkout & Payment:** Seamless ordering and payment process for customers.

**Admin Panel (Back-End)**
* **Order Management:** Real-time dashboard to view incoming orders and update order statuses (e.g., to "Served").
* **Inventory Control:** Comprehensive stock management system to track and update ingredient availability.

## 🛠️ Tech Stack

* **Front-End:** HTML5, CSS3, JavaScript
* **Back-End:** PHP
* **Database:** MySQL *(ถ้าคุณใช้ MySQL เก็บข้อมูล Stock/Order แนะนำให้ใส่เพิ่มไปด้วยครับ เพราะมักจะใช้คู่กับ PHP)*
